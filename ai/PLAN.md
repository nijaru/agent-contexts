# Strategic Plan: Agent Contexts

## Goal
Provide a widely-adopted, standardized organization pattern for AI coding agents across all tools and workflows.

## Milestones

- **Phase 1 (Q4 2024 - Q1 2025)**: Foundation ← CURRENT
  - Deliverables: Core PRACTICES.md, example templates, token optimization
  - Success criteria: Clean, minimal structure (<2000 words total), battle-tested in personal projects
  - Status: COMPLETE (efe7bbf adds PLAN.md + compaction rules)

- **Phase 2 (Q1-Q2 2025)**: Adoption & refinement
  - Deliverables: Community feedback integration, edge case documentation
  - Success criteria: Used by 3+ external projects, proven scalability patterns
  - Dependencies: Phase 1 stability proven

- **Phase 3 (Q2-Q3 2025)**: Tooling & automation
  - Deliverables: CLI helper scripts, template generators, validation tools
  - Success criteria: One-command project setup, automated compaction helpers
  - Dependencies: Phase 2 adoption validates need for tooling

## Critical Dependencies
- Phase 2 needs real-world usage data from diverse projects
- Phase 3 automation should only codify proven patterns, not speculative ones

## Out of Scope
- Language/framework-specific patterns (belongs in project CLAUDE.md)
- AI coding tutorials (AI already knows syntax)
- Comprehensive documentation (keep minimal by design)
- Integration with specific AI tools beyond file structure

## Success Metrics
- Adoption: External projects using this structure
- Maintenance: Low - structure should be stable and self-explanatory
- Token efficiency: ai/ files remain <500 lines per file on average
