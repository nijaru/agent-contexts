# Strategic Plan: Agent Contexts

## Goal
Provide a widely-adopted, standardized organization pattern for AI coding agents across all tools and workflows.

## Milestones

| Phase | Timeline | Status | Deliverables | Success Criteria |
|-------|----------|--------|--------------|------------------|
| **Phase 1** | Q4 2024 - Q1 2025 | COMPLETE | Core PRACTICES.md, templates, token optimization | <2000 words total, battle-tested |
| **Phase 2** | Q1-Q2 2025 | ← CURRENT | Community feedback, edge cases | 3+ external projects using structure |
| **Phase 3** | Q2-Q3 2025 | Planned | CLI helpers, template generators | One-command setup, auto-compaction |

## Critical Dependencies

| Dependency | Blocks | Reason |
|------------|--------|--------|
| Phase 1 stability proven | Phase 2 start | Need battle-tested foundation before community adoption |
| Real-world usage data | Phase 3 tooling | Automation should codify proven patterns, not speculative ones |

## Out of Scope
- Language/framework-specific patterns (belongs in project CLAUDE.md)
- AI coding tutorials (AI already knows syntax)
- Comprehensive documentation (keep minimal by design)
- Integration with specific AI tools beyond file structure

## Success Metrics

| Metric | Target | Current Status |
|--------|--------|----------------|
| Adoption | 10+ external projects | ~3 (personal projects) |
| Maintenance | <5 PRs/month | Stable, minimal changes needed |
| Token efficiency | <500 lines/file avg | ✅ All files <300 lines |
